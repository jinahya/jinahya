Basic Key Codes


The following virtual key codes must be supported by implementations that provide full support for key events.

  • VK_0 through VK_9
  • VK_A through VK_Z
  • VK_ENTER            
  • VK_BACK_SPACE       
  • VK_TAB               
  • VK_SPACE            
  • VK_LEFT              
  • VK_UP                
  • VK_RIGHT             
  • VK_DOWN
