10  /**
11   * An instance of this class may be used to represent a dynamic selection of video
12   * output configurations based upon specified input video.
13   * An instance of <code>DynamicVideoOutputConfiguration</code> would mainly be used
14   * to allow for the <code>HScreen</code> resolution and the video output port resolution
15   * to closely match the resolution of the input video, generally with the intention of
16   * letting the display monitor connected to the output port manage aspect ratio conversions.
17   * <p>
18   * Such configurations are only valid for the current 
19   * {@link HostSettings#getMainVideoOutputPort main} video output port for 
20   * a given <code>HScreen</code>.
21   * If a video output port is not the current <i>main</i> output port or 
22   * ceases to be the <i>main</i>, then this configuration setting SHALL be 
23   * effectively ignored and output resolution settings SHALL revert to an
24   * implementation-specific configuration.
25   * <p>
26   * Application of the input-to-output video resolution mapping described by an
27   * instance of <code>DynamicVideoOutputConfiguration</code> MAY result in configuration
28   * changes for the component <code>HScreenDevice</code>s of the relevant
29   * <code>HScreen</code> as if the output resolution were selected via a
30   * {@link FixedVideoOutputConfiguration static} configuration.
