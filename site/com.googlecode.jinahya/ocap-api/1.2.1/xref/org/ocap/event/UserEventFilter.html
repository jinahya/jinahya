8   /**
9    * Only one instance of the class that implements this interface can be 
10   * registered to EventManager via the EventManager.setUserEventFilter() 
11   * method. EventManager calls UserEventFilter.filterUserEvent() before a
12   * UserEvent is forwarded to any listening applications for events specified 
13   * by the UserEventRepository set via the EventManager.setFilteredRepository() 
14   * method.
15   */
16  public interface UserEventFilter
17  {
18      /**
19       * Called by the platform to filter user input events specified in the 
20       * UserEventRepository when the event is received. The EventFilter may 
21       * modify values in the UserEvent and/or change the applications to which 
22       * the event is forwarded. The UserEventFilter.filterUserEvent() returns 
23       * a UserEventAction instance which specifies the event to be forwarded
24       * and the list of AppIDs to which the event is forwarded. If it returns 
25       * null, the event is consumed, i.e., the event is not forwarded to any 
26       * applications.
27       *
28       *@param e the UserEvent which was received
29       *
30       *@return null if the event is to be terminated.  Otherwise, a 
31       * UserEventAction object that contains the event to distribute,
32       * and a list of AppIDs to which to distribute it.
33       */
34      public UserEventAction filterUserEvent(UserEvent e);
35  }
