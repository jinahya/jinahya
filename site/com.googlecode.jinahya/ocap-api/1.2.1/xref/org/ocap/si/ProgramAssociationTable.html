6   /**
7    * This interface represents an MPEG-2 Program Association Table (PAT).
8    * <p></p>
9    * For an Inband PAT, the getLocator() method defined in the SIElement
10   * interface shall return an org.ocap.net.OcapLocator instance corresponding to
11   * one of the following OCAP URL forms:
12   * <p></p>
13   * <tt>ocap://source_id</tt></br>
14   * <tt>ocap://f=frequency.program_number</tt>
15   * <p></p>
16   * The form returned must match the form of the OCAP URL
17   * passed to the previous call to
18   * <tt>ProgramAssociationTableManager.retrieveInBand()</tt>
19   * or
20   * <tt>ProgramAssociationTableManager.addInBandChangeListener()</tt>.
21   * <p></p>
22   * For an OOB PAT, the returned OcapLocator
23   * corresponds to the following OCAP URL form:
24   * <p></p>
25   * <tt>ocap://oobfdc.program_number</tt>
26   * <p></p>
27   * The getServiceInformationType() method defined in the SIElement interface 
28   * shall return ServiceInformationType.UNKNOWN.
29   */
