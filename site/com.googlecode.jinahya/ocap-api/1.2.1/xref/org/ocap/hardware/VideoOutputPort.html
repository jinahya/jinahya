3   /**
4    * An object of this class represents an analog or digital video output of
5    * the OCAP terminal. If the type is the analog video output, it is mapped
6    * to a physical pin plug of the video output. If the type is the digital serial
7    * output, it is mapped to a physical pin plug of the video output. If the type
8    * is the digital bus output, it is mapped to a bus node that has several output
9    * ports. For example, if the type is the 1394 bus, the VideoOutputPort instance
10   * represents not the 1394 port (physical pin plug) but the 1394 node that has
11   * several 1394 ports.
12   * <p>
13   * An application cannot construct an instance of this class directly.
14   * Instead, the Host.getVideoOutputPorts() method is used to obtain a
15   * java.util.Enumeration of references to VideoOutputPort instances.
16   * The Enumeration.nextElement() method can be used to obtain references to
17   * individual VideoOutputPort instances.
18   * </p>
19   * The video port is a scarce resource, but the resource management framework is
20   * not applied. At most only one 1394 connection is available for a single OCAP
21   * implementation. If some applications call VideoOutputPort.select1394sink(),
22   * only the last call is effective. The other calls are ignored or disconnected
23   * without any notification.
24   */
