225     /**
226      * <p>
227      * This method returns a possible attribute values applied to an 
228      * closed-captioning text on a screen. 
229      * Note that the possible font attribute may be different from the 
230      * possible font for Java application since the closed-captioning module 
231      * may be implemented by native language. 
232      * </p>
233      *
234      * @return  an array of possible attribute values of an closed-captioning 
235      *          text corresponding to the specified attribute parameter. 
236      *          <ul>
237      *          <li> If the attribute parameter is CC_ATTRIBUTE_PEN_FG_COLOR 
238      *          or CC_ATTRIBUTE_PEN_BG_COLOR, an array of java.awt.Color that 
239      *          represents possible font color returns. The Color.getString() 
240      *          shall return a text expression of its color to show a user. 
241      *          <li> If the attribute parameter is CC_ATTRIBUTE_PEN_FG_OPACITY 
242      *          or CC_ATTRIBUTE_PEN_BG_OPACITY, an array of constants that 
243      *          represents possible opacity returns. The opacity constants has 
244      *          a prefix of CC_OPACITY_. 
245      *          <li> If the attribute parameter is CC_ATTRIBUTE_FONT_STYLE, an 
246      *          array of String that represents possible font style returns. 
247      *          It is recommended that the String is one of font style defined 
248      *          in EIA-708-B but not restricted to it. The host device can 
249      *          provide a new style. 
250      *          <li> If the attribute parameter is CC_ATTRIBUTE_PEN_SIZE, an 
251      *          array of constants that represents possible pen size returns. 
252      *          The pen size constants has a prefix of CC_PEN_SIZE_. 
253      *          <li> If the attribute parameter is CC_ATTRIBUTE_FONT_ITALICIZED, 
254      *          an array of possible Integer value (YES=1, NO=0) returns. 
255      *          I.e., if the host can select a plane font or an italicized font, 
256      *          an array of [0, 1] (or [1, 0]) returns. If the host only supports 
257      *          a plane font, [0] returns. 
258      *          <li> If the attribute parameter is CC_ATTRIBUTE_FONT_UNDERLINE, 
259      *          an array of possible Integer value (YES=1, NO=0) returns. 
260      *          See also the CC_ATTRIBUTE_FONT_ITALICIZED description. 
261      *          <li> If the attribute parameter is 
262      *          CC_ATTRIBUTE_WINDOW_FILL_COLOR, an array of java.awt.Color that 
263      *          represents possible window fill color returns. 
264      *          The Color.getString() shall return a text expression of its 
265      *          color to show a user. 
266      *          <li> If the attribute parameter is 
267      *          CC_ATTRIBUTE_WINDOW_FILL_OPACITY an array of constants that 
268      *          represents possible opacity returns. The opacity constants has 
269      *          a prefix of CC_OPACITY_. 
270      *          <li> If the attribute parameter is 
271      *          CC_ATTRIBUTE_WINDOW_BORDER_TYPE an array of constants that 
272      *          represents possible border type returns. The border type 
273      *          constants has a prefix of CC_BORDER_. 
274      *          <li> If the attribute parameter is 
275      *          CC_ATTRIBUTE_WINDOW_BORDER_COLOR, an array of java.awt.Color 
276      *          that represents possible window border color returns. 
277      *          The Color.getString() shall return a text expression of its 
278      *          color to show a user. 
279      *          </ul>

282      *          One of constants that has a CC_ATTRIBUTE_ prefix shall be
