9   /**
10   * An OCAP-J application can set and get a preferred attribute value of an 
11   * EAS alert text. 
12   * The capability method provides possible attribute values. And also, an 
13   * OCAP-J application can set an EASHandler to be notified of a private 
14   * descriptor indicating an alternative audio, if the alert_priority=15 but 
15   * no audio specified by SCTE 18 is available.
16   * The OCAP implementation may throw an exception if a specified preference 
17   * doesn't conform to FCC rules or the SCTE 18 specification. For example, 
18   * some color combinations could make text unreadable. 
19   * See http://www.fcc.gov/eb/eas/ for FCC rules. 
20   * </p><p>

156     /**
157      * <p>
158      * This method sets a preferred attribute values applied to an EAS alert 
159      * text on a screen. If the specified attribute value is invalid, i.e., 
160      * the value is not included in the return value of the 
161      * {@link #getEASCapability} method, this method doesn't change current 
162      * attribute value and throw an exception. If the specified attribute is 
163      * EAS_ATTRIBUTE_FONT_OPACITY or EAS_ATTRIBUTE_BACK_OPACITY, this method accepts   
164      * any Float value and the OCAP implementation tries to apply it. The OCAP 
165      * implementation may not able to apply the specified opacity exactly. 
166      * </p><p>
167      * Note that even if the application could set a preference 
168      * successfully, the OCAP implementation may not apply it to an 
169      * EAS message text on a screen if a conflict with FCC rule 
170      * or SCTE 18 specification is found while displaying process. 
171      * </p>
172      *
173      * @param attribute  an array of constants that has a prefix of 
174      *          EAS_ATTRIBUTE_ to specify an attribute. 
175      *
176      * @param value  an array of preferred attribute values to be set to an 
177      *          alert text. The i-th item of the value array corresponds to 
178      *          the i-th item of the attribute array. 
179      *          <ul>
180      *          <li> If the attribute parameter is EAS_ATTRIBUTE_FONT_COLOR, 
181      *          an instance of java.awt.Color that represents preferred font 
182      *          color. 
183      *          <li> If the attribute parameter is EAS_ATTRIBUTE_FONT_STYLE, 
184      *          an String that represents preferred font style. 
185      *          <li> If the attribute parameter is EAS_ATTRIBUTE_FONT_FACE, 
186      *          an String that represents preferred font face name. 
187      *          <li> If the attribute parameter is EAS_ATTRIBUTE_FONT_SIZE, 
188      *          an Integer of preferred font size. 
189      *          <li> If the attribute parameter is EAS_ATTRIBUTE_BACK_COLOR, 
190      *          an instance of java.awt.Color that represents preferred 
191      *          background color. 
192      *          <li> If the attribute parameter is EAS_ATTRIBUTE_FONT_OPACITY, 
193      *          an Float of preferred font opacity. 
194      *          <li> If the attribute parameter is EAS_ATTRIBUTE_BACK_OPACITY, 
195      *          an Float of preferred background opacity. 
196      *          </ul>
197      *
198      * @throws IllegalArgumentException  if the attribute is out of range 
199      *             or the value is not a possible value or if the specified 
200      *             preference conflicts with FCC rules or SCTE 18. For example, 
201      *             an EAS message is invisible since same color is specified to 
202      *             a font and background. Criteria of visibility depends on 
203      *             a manufacturer or a display device etc.
