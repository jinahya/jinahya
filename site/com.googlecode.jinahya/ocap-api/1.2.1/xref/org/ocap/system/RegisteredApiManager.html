30      /**
31       * Registers an API with the implementation.
32       * <p>
33       * If the name and version number matches an API already registered, this function does
34       * nothing (successfully). Matches for both name and version are based on exact
35       * case sensitive comparisons.
36       * <p>
37       * If the name matches an API already registered,
38       * and the version number is different, the implementation SHALL:
39       * remove the existing API before installing the new API.  The removal SHALL obey
40       * the semantics specified for the unregister() method.  If the installation fails then
41       * the previously registered API SHALL NOT be removed.  The removal of the
42       * previous API and installation of the new API SHALL be one atomic operation.
43       * (Note: This implies that the terminal MUST download and authenticate all files required
44       * for the new API, and only if this succeeds can it then remove the old API &
45       * install the new API.  Application authors that do not need this behavior
46       * should note that unregistering the old API before registering a new version
47       * may reduce the memory usage of this operation and is strongly recommended).
48       * <p>
49       * Paths in the SCDF are relative to the directory containing the SCDF.
50       * The priority field specified in the SCDF is ignored.
51       *
52       * @param name Name of the registered API.
53       * @param version Version of the registered API.
54       * @param scdf Path to the shared classes descriptor file.
55       * @param storagePriority Storage priority of classes in the SCDF.
56       *
57       * @throws IllegalArgumentException if storagePriority is not a valid value as
58       *      defined in chapter 12.
59       * @throws IllegalStateException if the API to be updated is in use by any application.
60       * @throws java.io.IOException if the SCDF or any file listed in it does not exist,
61       *       cannot be loaded, or are not correctly signed.  Also thrown if the SCDF
62       *       is not the correct format and cannot be parsed.
63       * @throws SecurityException if the calling application does not have
64       *      MonitorAppPermission("registeredapi.manager").
65       */
66      public abstract void register(String name,
67                                    String version,
68                                    File scdf,
69                                    short storagePriority)
