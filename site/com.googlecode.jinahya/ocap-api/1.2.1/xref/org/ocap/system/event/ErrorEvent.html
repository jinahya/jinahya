8   /**
9    * This class represents an event returned by the system when an uncaught exception or
10   * implementation error is encountered, or by an application that wishes to log an error or
11   * an informational event.  Error event type codes are defined in this class.
12   * Applications may use the error type codes in this class or proprietary class
13   * codes that are understood by the network.
14   * <P>
15   * The class takes a {@link java.lang.Throwable} object parameter in one constructor, but the
16   * Throwable object cannot be returned from this class due to implementation and security
17   * issues.  The Throwable object attributes (i.e., message and stacktrace) can be retrieved
18   * from this class by calling corresponding get methods, which in-turn call the Throwable
19   * object get methods.  However, the implementation MUST NOT allow the Throwable object get
20   * methods to block indefinitely when called and MUST NOT wait longer than 30 seconds for them
