12  /**
13     The {@link org.havi.ui.HRangeValue HRangeValue} is a user interface
14     component used to display a value within a fixed range (as
15     org.havi.ui.HStaticRange HStaticRange}) which enables a user to
16     navigate to and alter the value of it, i.e. it can have the input
17     focus and it can be adjusted. By default it uses the {@link
18     org.havi.ui.HRangeLook HRangeLook} to render itself.
19  
20    <hr>

22    parameters are not used, then the constructor should use the default
