11  /**
12     An {@link org.havi.ui.event.HItemEvent} event is used to
13     interact with a component implementing the {@link
14     org.havi.ui.HSelectionInputPreferred}
15     interface as follows:
16  
17     <p><ul>
18  
19     <li> An {@link org.havi.ui.event.HItemEvent} event may be sent
20     from the HAVi system to the component to change the state of the
21     component, and the selection set held by the component.
22  
23     <li> An {@link org.havi.ui.event.HItemEvent} event is sent
24     from the component to all registered {@link
25     org.havi.ui.event.HItemListener}s when a change to
26     the component state or selection set occurs.
27     
28     </ul><p>
29  
30     The {@link org.havi.ui.event.HItemEvent} event class therefore
31     provides support for selecting and clearing individual elements,
32     selecting or clearing all elements and managing the currently
33     focused element, i.e. the item whose selection state is toggled by
34     some user interaction. 
35  
36     <p>
37     All interoperable HAVi components which expect to receive {@link
38     org.havi.ui.event.HItemEvent} events should implement the
39     {@link org.havi.ui.HSelectionInputPreferred} interface.   
40  
41     @see org.havi.ui.HListGroup
42  
43   */
44
