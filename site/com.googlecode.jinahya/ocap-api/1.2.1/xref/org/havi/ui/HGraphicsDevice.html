247     /**
248      * Set the graphics configuration for the device.
249      * <p>
250      * An application is only allowed to call this method after it
251      * reserved the device explicitly and subject to the security
252      * policy of the platform. Subject to this, the following rules
253      * determine whether this method can succeed.
254      * <ul>
255      * <li>If an application tries to select a configuration which is
256      * not valid for that device at that time or when the device is in a
257      * particular mode then an HConfigurationException shall be thrown.
258      * <li>If the application selects a configuration that is not
259      * conflicting with the configurations of all the other devices on
260      * the same {@link org.havi.ui.HScreen} then that
261      * configuration is selected.
262      * <li>If an application tries to select a configuration that
263      * conflicts with a configuration of another device on this {@link
264      * org.havi.ui.HScreen} which this application cannot
265      * control due to platform security policy then this method shall
266      * fail with a <code>SecurityException</code>.
267      * <li>If an application tries to select a configuration that
268      * conflicts with a configuration of another device on this {@link
269      * org.havi.ui.HScreen} which this application cannot
270      * control due to another application owning the right to control
271      * that device and the platform not giving that right to this
272      * application then this method shall fail with an {@link
273      * org.havi.ui.HPermissionDeniedException}.
274      * <li>If an application tries to select a configuration that
275      * conflicts with a configuration of another device which this
276      * application can control due to either this or no application
277      * having reserved the device, then the configuration of the other
278      * device is automatically changed. There is no change in the
279      * resource ownership of the other device.
280      * </ul>     
281      * <p>    
282      * Applications can prevent or limit changes to configurations of
283      * other, not intended, devices by using constants {@link
284      * HScreenConfigTemplate#ZERO_GRAPHICS_IMPACT}, {@link
285      * HScreenConfigTemplate#ZERO_VIDEO_IMPACT} and {@link
286      * HScreenConfigTemplate#ZERO_BACKGROUND_IMPACT}
287      * in their configuration templates. The first
288      * one will cause no changes to already running graphical
289      * applications. This means that no changes may be applied to the
290      * graphics device. Similarly the second constant will result in
291      * no changes to the video device.
292      * <p> 
293      * Any modifications made to the configurations of other devices
294      * shall be reflected by the API for the devices concerned. The
295      * {@link org.havi.ui.HGraphicsDevice#getCurrentConfiguration}
296      * method for those devices shall return
297      * the new configuration.
298      * <p>
299      * On successful change to the specified configuration, the device
300      * shall fire one or more {@link
301      * org.havi.ui.event.HScreenConfigurationEvent} for all listeners that are
302      * currently registered (if the criteria for receiving such an
303      * event has been satisfied). If the new configuration differs by
304      * more than one characteristic from the previous configuration
305      * then the device may fire one or more {@link
306      * org.havi.ui.event.HScreenConfigurationEvent}.
307      * This behavior is implementation specific.
308      * <p>
309      * If an attempt is made to set the identical configuration to
310      * that which is current, then no such event(s) shall be fired.
311      *
312      * @param hgc the HGraphicsConfiguration 
313      * to which this device should be set.
314      * @return A boolean indicating whether the configuration could be
315      * applied successfully. If the configuration could not be applied
316      * successfully, the configuration after this method may not match
317      * the configuration of the device prior to this method being
318      * called --- applications should take steps to determine whether
319      * a partial change of settings has been made.
320      * @exception SecurityException if the application does not have
321      * sufficient rights to set the configuration for this device.
322      * @exception HPermissionDeniedException
323      * if the application does not
324      * currently have the right to set the configuration for this
325      * device.
326      * @exception HConfigurationException
327      * if the specified configuration is not
