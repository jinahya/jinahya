10  /**
11     An {@link org.havi.ui.event.HFocusEvent HFocusEvent} event is used to
12     interact with a component implementing the {@link
13     org.havi.ui.HNavigationInputPreferred HNavigationInputPreferred}
14     interface as follows:
15  
16     <p><ul>
17  
18     <li> An {@link org.havi.ui.event.HFocusEvent HFocusEvent} event may be sent
19     from the HAVi system to the component to inform the component that
20     it has gained or lost the input focus, or that it should transfer
21     focus to another component.
22  
23     <li> An {@link org.havi.ui.event.HFocusEvent HFocusEvent} event is sent
24     from the component to all registered {@link
25     org.havi.ui.event.HFocusListener HFocusListener} listeners whenever
26     the component focus status changes. 
27     
28     </ul><p>
29  
30     Note that because the underlying focus mechanism is based on AWT,
31     focus transfer events do not guarantee that another component will
32     actually get focus, or that the current component will lose focus.
33  
34     <p>
35     All interoperable HAVi components which expect to receive {@link
36     org.havi.ui.event.HFocusEvent HFocusEvent} events must implement the
37     {@link org.havi.ui.HNavigationInputPreferred
38     HNavigationInputPreferred} interface.
39     
40   */
