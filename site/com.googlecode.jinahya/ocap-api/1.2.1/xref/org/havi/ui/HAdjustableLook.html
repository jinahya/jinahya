6   /**
7      The {@link org.havi.ui.HAdjustableLook HAdjustableLook} interface is implemented by
8      all platform looks which support orientable components (i.e. those
9      components which implement the {@link org.havi.ui.HOrientable
10     HOrientable} interface. The following platform looks shall
11     implement this interface:
12  
13     <p><ul>
14     <li> {@link org.havi.ui.HRangeLook HRangeLook}
15     <li> {@link org.havi.ui.HListGroupLook HListGroupLook}
16     </ul><p>
17  
18     The {@link org.havi.ui.HAdjustableLook HAdjustableLook} interface supports pointer
19     based systems by providing a mechanism of &quot;hit-testing&quot;
20     which allows the {@link org.havi.ui.HOrientable HOrientable} component to 
21     determine which part of the on-screen representation has been clicked in, and 
22     to adjust its internal orientable value accordingly (for 
23     {@link org.havi.ui.HListGroup HListGroup}, the orientable value is the scroll  
24     position of the {@link org.havi.ui.HListGroup HListGroup}).
25  
26

28     The diagram below shows one possible on-screen representation of an
29     {@link org.havi.ui.HOrientable HOrientable} component,
30     with {@link org.havi.ui.HOrientable#ORIENT_LEFT_TO_RIGHT
31     ORIENT_LEFT_TO_RIGHT} orientation.

37     <p> 
38     HLook implementations which implement {@link org.havi.ui.HAdjustableLook
39     HAdjustableLook} may use the {@link
40     org.havi.ui.HOrientable#getOrientation getOrientation} method
41     to determine the appropriate constant to return from {@link
42     org.havi.ui.HAdjustableLook#hitTest hitTest} since the correct
43     constant is dependent on the orientation of the component.
44  
45     <p> 
46     It is a valid implementation option to return {@link

48     {@link org.havi.ui.HAdjustableLook#hitTest hitTest} method in all cases. 
49  
50     <p>
51     It is a valid implementation option to never return {@link
52     org.havi.ui.HAdjustableLook#ADJUST_BUTTON_LESS ADJUST_BUTTON_LESS}
53     and {@link org.havi.ui.HAdjustableLook#ADJUST_BUTTON_MORE

129     /**
