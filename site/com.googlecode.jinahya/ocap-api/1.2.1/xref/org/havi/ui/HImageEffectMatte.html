12  /**
13     The {@link org.havi.ui.HImageEffectMatte HImageEffectMatte} class
14     represents a matte that varies over both space and time, it is
15     specified as a sequence of image masks.
16     
17     <p>
18     The data for any HImageEffectMatte may be changed &quot;on the fly&quot;
19     using the {@link org.havi.ui.HImageEffectMatte#setMatteData setMatteData} method. 
20     
21     However, some implementations may be asynchronously referencing
22     their content (i.e. through a separate implementation-specific
23     animation thread).  Therefore the following restrictions apply to
24     the {@link org.havi.ui.HImageEffectMatte#setMatteData setMatteData} method:
25     
26     <p><ul>                 
27     <li>The method must be synchronized with any implementation-specific
28     animation thread such that content cannot be changed while a
29     different thread is using it. 
30     <li>If the animation was running the method should stop the
31     animation in a synchronized manner before changing content.  
32     <li>The method should reset the animation to a starting position
33     defined by the current play mode. The repeat count of the animation
34     should be reset to 0.  
35     <li>If the animation was running the method should start the
36     animation.  
37     </ul>
