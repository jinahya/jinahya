8   /**
9      The {@link org.havi.ui.HVideoConfigTemplate}
10     class is used to obtain a valid {@link
11     org.havi.ui.HVideoConfiguration}.  An
12     application instantiates one of these objects and then sets all
13     non-default attributes as desired. The object is then passed to the
14     {@link org.havi.ui.HVideoDevice#getBestConfiguration} method found in the
15     {@link org.havi.ui.HVideoDevice} class.  If possible,
16     a valid {@link org.havi.ui.HVideoConfiguration}
17     is returned which meets or exceeds the requirements set in
18     the {@link org.havi.ui.HVideoConfigTemplate}.
19  
20     <p>

22     properties of video configurations which may be requested by
23     applications.

53      /**
54       * A value for use in the preference field of the {@link
55       * org.havi.ui.HScreenConfigTemplate#setPreference setPreference}
56       * and {@link org.havi.ui.HScreenConfigTemplate#getPreferencePriority
57       * getPreferencePriority} methods in the {@link
58       * org.havi.ui.HVideoConfigTemplate HVideoConfigTemplate} that
59       * indicates that the device configuration supports the display of
60       * graphics in addition to video streams.  This display includes
61       * both configurations where the video pixels and graphics pixels
62       * are fully aligned (same size) as well as configurations where
63       * they are displayed together but where a more complex
64       * relationship exists between the two pixel coordinate
65       * spaces. The graphics configuration for mixing is specified as an
66       * {@link org.havi.ui.HGraphicsConfiguration
67       * HGraphicsConfiguration}.<p>
68       * This preference is used by the platform as a constraint in
69       * selecting configurations. Templates generated by the platform
70       * and then returned to applications (e.g. from a
71       * <code>getConfigTemplate</code> method) shall not have this
72       * preference filled in by the platform.     
73       */
74      public static final int GRAPHICS_MIXING = 0x0F;
