11  /**
12    This class is able to describe the representation of an event
13    generator as a string, color or symbol (such as a triangle, '>', for
14    'play'). This allows an application to describe a button on an input
15    device correctly for a given platform.
16  
17    <p> 
18  
19    The particular text, color, or symbol can be determined by calling
20    the {@link org.havi.ui.event.HEventRepresentation#getString
21    getString}, {@link org.havi.ui.event.HEventRepresentation#getColor
22    getColor} or {@link org.havi.ui.event.HEventRepresentation#getSymbol
23    getSymbol} methods respectively. All available events should return
24    a valid text representation from the {@link
25    org.havi.ui.event.HEventRepresentation#getString getString} method.
26  
27    <p> 
28    If supported the six colored key events
29    (<code>VK_COLORED_KEY_0</code> thru <code>VK_COLORED_KEY_5</code>)
30    must also be represented by a color, i.e. the {@link
31    org.havi.ui.event.HEventRepresentation#getColor getColor} method
32    must return a valid <code>java.awt.Color</code> object.
33
