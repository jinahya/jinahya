This sound will start to be played when an object implementing this interface gains focus. It is not guaranteed to be played to completion. If the object implementing this interface loses focus before the audio completes playing, the audio will be truncated.  Applications wishing to ensure the audio is always played to completion must implement special logic to slow down the focus transitions. <p> By default, an  {@link org.havi.ui.HNavigable  HNavigable}  object does not have any gain focus sound associated with it. <p> Note that the ordering of playing sounds is dependent on the order of the focus lost, gained events.
