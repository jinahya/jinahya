3   /**
4    * This class represents an MPEG-2 Transport Stream with its 
5    * associated Service Information (SI) as known to a decoder.
6    * <p>
7    * A TransportStream object models a specific transport stream that can be 
8    * accessed through a specific network interface. This implies that a Transport
9    * Stream object has implicitly a connection to a specific network interface. 
10   * Thus, if two or more network interfaces can deliver the same transport 
11   * stream, this will be modeled by two or more separate TransportStream objects. 
12   * A network interface does not need to be tuned to a transport stream if an 
13   * application wants to use the corresponding TransportStream object.
14   * <p>
15   * If the corresponding network interface is currently tuned to the 
16   * TransportStream, then an application can query the TransportStream for the 
17   * services it contains, and the services for which elementary streams it 
18   * contains. If the corresponding network interface is not currently tuned to 
19   * the TransportStream, then the application can query the TransportStream for
20   * the services it contains. If the STB has cached the required information 
21   * it can return it, otherwise it should return null. If an application queries
22   * a Service object for elementary stream information and the corresponding 
23   * TransportStream is not currently tuned to, then the Service object should 
24   * return null.
25   * <p>
26   * If an application has two references to a TransportStream object and those 
27   * TransportStream objects model the same transport stream coming from the same
28   * network interface, then the equals() method (inherited from java.lang.Object) 
29   * return true when comparing both TransportStream objects. The references 
30   * themselves are not necessarily the same, although they may be.
31   * <p>
32   * Note: If an application wants to know to which network interface a 
33   * TransportStream object is connected to, it should use the Tuning API if it 
34   * is available.
35   */
