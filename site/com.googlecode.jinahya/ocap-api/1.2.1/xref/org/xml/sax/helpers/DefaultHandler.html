43  /**
44   * Default base class for SAX2 event handlers.
45   *
46   * <p>This class is available as a convenience base class for SAX2
47   * applications: it provides default implementations for all of the
48   * callbacks in the four core SAX2 handler classes:</p>
49   *
50   * <ul>
51   * <li>{@link org.xml.sax.EntityResolver EntityResolver}</li>
52   * <li>{@link org.xml.sax.DTDHandler DTDHandler}</li>
53   * <li>{@link org.xml.sax.ContentHandler ContentHandler}</li>
54   * <li>{@link org.xml.sax.ErrorHandler ErrorHandler}</li>
55   * </ul>
56   *
57   * <p>Application writers can extend this class when they need to
58   * implement only part of an interface; parser writers can
59   * instantiate this class to provide default handlers when the
60   * application has not supplied its own.</p>
61   *
62   * @since SAX 2.0
63   * @author David Megginson, 
64   * @see org.xml.sax.EntityResolver
