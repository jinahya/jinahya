40  /**
41   *  The <code>EventListener</code> interface is the primary method for 
42   * handling events. Users implement the <code>EventListener</code> interface 
43   * and register their listener on an <code>EventTarget</code. The users 
44   * should also remove their 
45   * <code>EventListener</code> from its <code>EventTarget</code> after they 
46   * have completed using the listener. 
47   * <p> Copying a <code>Node</code> with the <code>Node.cloneNode</code>
48   * method does not copy the event listeners attached to it. Event
49   * listeners must be attached to the newly created <code>Node</code>
50   * afterwards if so desired.
51   * <p> Moving a <code>Node</code>, with methods <code>Document.adoptNode</code>
52   * or <code>Node.appendChild</code>, does not affect the event listeners 
53   * attached to it.
54   * <p> The EventListener behavior follows the description in section
55   * 1.2.2 of the DOM Level 3 Events specification. 
56   * <p>See also the 
57   * <a href='http://www.w3.org/TR/DOM-Level-3-Events'>
58   * Document Object Model (DOM) Level 3 Events Specification</a> and .
59   * @since DOM Level 2
60   */
61  public interface EventListener {
62      /**
63       *  This method is called whenever an event occurs of the type for which 
64       * the <code> EventListener</code> interface was registered. 
65       * @param evt The <code>Event</code> contains contextual information 
66       *   about the event.
67       */
68      public void handleEvent(Event evt);

70  }
