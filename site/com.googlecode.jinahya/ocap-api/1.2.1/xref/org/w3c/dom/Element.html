40  /**
41   * The <code>Element</code> interface represents an element in an HTML or XML 
42   * document. Elements may have attributes associated with them; since the 
43   * <code>Element</code> interface inherits from <code>Node</code>, the 
44   * generic <code>Node</code> interface attribute <code>attributes</code> may 
45   * be used to retrieve the set of all attributes for an element. There are 
46   * methods on the <code>Element</code> interface to retrieve either an 
47   * <code>Attr</code> object by name or an attribute value by name. In XML, 
48   * where an attribute value may contain entity references, an 
49   * <code>Attr</code> object should be retrieved to examine the possibly 
50   * fairly complex sub-tree representing the attribute value. On the other 
51   * hand, in HTML, where all attributes have simple string values, methods to 
52   * directly access an attribute value can safely be used as a convenience.
53   * <p><b>Note:</b> In DOM Level 2, the method <code>normalize</code> is 
54   * inherited from the <code>Node</code> interface where it was moved.
55   * <p>See also the <a href='http://www.w3.org/TR/2000/REC-DOM-Level-2-Core-20001113'>Document Object Model (DOM) Level 2 Core Specification</a> and the <a href='http://www.w3.org/TR/2004/REC-DOM-Level-3-Core-20040407'>Document Object Model (DOM) Level 3 Core Specification</a>.
56   */
57  public interface Element extends Node {
58      /**
59       * The name of the element. If <code>Node.localName</code> is different
60       * from <code>null</code>, this attribute is a qualified name.
61       * For example, in: 
62       * <pre> &lt;elementExample id="demo"&gt; ... 
63       * &lt;/elementExample&gt; , </pre>
64       *  <code>tagName</code> has the value
65       * <code>"elementExample"</code>. Note that this is case-preserving in
66       * XML, as are all of the operations of the DOM. The HTML DOM returns
67       * the <code>tagName</code> of an HTML element in the canonical
68       * uppercase form, regardless of the case in the source HTML document.
69       * @return a String containing the name of the element
70       */
71      public String getTagName();

73      /**
74       * Retrieves an attribute value by name.
75       * @param name The name of the attribute to retrieve.
76       * @return The <code>Attr</code> value as a string, or the empty string 
77       *   if that attribute does not have a specified or default value.
78       */
79      public String getAttribute(String name);

81      /**
82       * Adds a new attribute. If an attribute with that name is already present 
83       * in the element, its value is changed to be that of the value 
84       * parameter. This value is a simple string; it is not parsed as it is 
85       * being set. So any markup (such as syntax to be recognized as an 
86       * entity reference) is treated as literal text, and needs to be 
87       * appropriately escaped by the implementation when it is written out. 
88       * In order to assign an attribute value that contains entity 
89       * references, the user must create an <code>Attr</code> node plus any 
90       * <code>Text</code> and <code>EntityReference</code> nodes, build the 
91       * appropriate subtree, and use <code>setAttributeNode</code> to assign 
92       * it as the value of an attribute.
93       * <br>To set an attribute with a qualified name and namespace URI, use 
94       * the <code>setAttributeNS</code> method.
95       * @param name The name of the attribute to create or alter.
96       * @param value Value to set in string form.
97       * @exception DOMException
98       *   INVALID_CHARACTER_ERR: Raised if the specified name is not an XML 
99       *   name according to the XML version in use (1.0 for JSR 280).
100      *   <br>NO_MODIFICATION_ALLOWED_ERR: Raised if this node is readonly.
101      */
102     public void setAttribute(String name, 
103                              String value)
104                              throws DOMException;

116      *   NO_MODIFICATION_ALLOWED_ERR: Raised if this node is readonly.

133     /**
134      * Adds a new attribute node. If an attribute with that name (
135      * <code>nodeName</code>) is already present in the element, it is 
136      * replaced by the new one. Replacing an attribute node by itself has no 
137      * effect.
138      * <br>To add a new attribute node with a qualified name and namespace 
139      * URI, use the <code>setAttributeNodeNS</code> method.
140      * @param newAttr The <code>Attr</code> node to add to the attribute list.
141      * @return If the <code>newAttr</code> attribute replaces an existing 
142      *   attribute, the replaced <code>Attr</code> node is returned, 
143      *   otherwise <code>null</code> is returned.
144      * @exception DOMException
145      *   WRONG_DOCUMENT_ERR: Raised if <code>newAttr</code> was created from a 
146      *   different document than the one that created the element.
147      *   <br>NO_MODIFICATION_ALLOWED_ERR: Raised if this node is readonly.
148      *   <br>INUSE_ATTRIBUTE_ERR: Raised if <code>newAttr</code> is already an 
149      *   attribute of another <code>Element</code> object. The DOM user must 
150      *   explicitly clone <code>Attr</code> nodes to re-use them in other 
151      *   elements.
152      */
153     public Attr setAttributeNode(Attr newAttr)
154                                  throws DOMException;

203      * namespace URI is already present on the element, its prefix is 
204      * changed to be the prefix part of the <code>qualifiedName</code>, and 
205      * its value is changed to be the <code>value</code> parameter. This 
206      * value is a simple string; it is not parsed as it is being set. So any 
207      * markup (such as syntax to be recognized as an entity reference) is 
208      * treated as literal text, and needs to be appropriately escaped by the 
209      * implementation when it is written out. In order to assign an 
210      * attribute value that contains entity references, the user must create 
211      * an <code>Attr</code> node plus any <code>Text</code> and 
212      * <code>EntityReference</code> nodes, build the appropriate subtree, 
213      * and use <code>setAttributeNodeNS</code> or 
214      * <code>setAttributeNode</code> to assign it as the value of an 
215      * attribute.
216      * <br>Per [<a href='http://www.w3.org/TR/1999/REC-xml-names-19990114/'>XML Namespaces</a>]
217      * , applications must use the value <code>null</code> as the 
218      * <code>namespaceURI</code> parameter for methods if they wish to have 
219      * no namespace.
220      * @param namespaceURI The namespace URI of the attribute to create or 
221      *   alter.
222      * @param qualifiedName The qualified name of the attribute to create or 
223      *   alter.
224      * @param value The value to set in string form.
225      * @exception DOMException
226      *   INVALID_CHARACTER_ERR: Raised if the specified qualified name is not 
227      *   an XML name according to the XML version in use (1.0 for JSR 280).
228      *   <br>NO_MODIFICATION_ALLOWED_ERR: Raised if this node is readonly.
229      *   <br>NAMESPACE_ERR: Raised if the <code>qualifiedName</code> is 
230      *   malformed per the Namespaces in XML specification, if the 
231      *   <code>qualifiedName</code> has a prefix and the 
232      *   <code>namespaceURI</code> is <code>null</code>, if the 
233      *   <code>qualifiedName</code> has a prefix that is "xml" and the 
234      *   <code>namespaceURI</code> is different from 
235      *   "<a href='http://www.w3.org/XML/1998/namespace'>
236      *   http://www.w3.org/XML/1998/namespace</a>", if the 
237      *   <code>qualifiedName</code> or its prefix is "xmlns" and the 
238      *   <code>namespaceURI</code> is different from 
239      *   "<a href='http://www.w3.org/2000/xmlns/'>http://www.w3.org/2000/xmlns/</a>", 
240      *   or  if the <code>namespaceURI</code> is 
241      *   "<a href='http://www.w3.org/2000/xmlns/'>http://www.w3.org/2000/xmlns/</a>"
242      *   and neither the <code>qualifiedName</code> nor its prefix is "xmlns".
