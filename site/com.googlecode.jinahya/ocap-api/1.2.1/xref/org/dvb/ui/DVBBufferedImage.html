12  /**
13   *
14   * The <code>DVBBufferedImage</code> subclass describes an {@link Image} with
15   * an accessible buffer of image data.
16   * The DVBBufferedImage is an adapter class for java.awt.image.BufferedImage. 
17   * It supports two different platform dependent sample models TYPE_BASE and TYPE_ADVANCED.
18   * Buffered images with the TYPE_BASE have the same sample model as the on screen graphics buffer, thus
19   * TYPE_BASE could be CLUT based. TYPE_ADVANCED has a direct color model but it is not specified
20   * how many bits are used to store the different color components.
21   * By default, a new DVBBufferedImage is transparent. All alpha values are set to 0;
22   * Instances of DVBBufferedImage shall be considered to be off-screen images for the 
23   * purpose of the inherited method <code>Image.getGraphics</code>.
24   * @since MHP 1.0
25   */
26

38  	 *	in the TYPE_INT_ARGB color model as well.
