3   /**
4    * This class is for return channel set-up permissions. An RCPermission 
5    * contains a name, but no actions list.
6    * <p>
7    * The permission name can be "target:default", which indicates
8    * the permission to use the default connection parameters.
9    * <p>
10   * The permission name can also be "target:&lt;phone number>", which
11   * indicates the permission to use the specified phone number in
12   * the connection set-up (ConnectionRCInterface.setTarget(ConnectionParameters) 
13   * method).
14   * <p>
15   * A wildcard may be used at the end of the permission name. In that
16   * case, all phone numbers starting with the number before the wildcard
17   * are included in the permission. 
18   * A "+" may be used at the start of the phone number to indicate a phone number 
19   * including the international country code.
20   * <p>
21   * Examples:
22   * <UL>
23   * <LI>
24   * target:0206234342  (Permission to dial the specified phone number)
25   * </LI>
26   * <LI>
27   * target:020*  (Permission to dial phone numbers starting with 020)
28   * </LI>
29   * <LI>
30   * target:*  (Permission to dial all phone numbers, including the default)
31   * </LI>
32   * </UL>
33   * <p>
34   * Note: ConnectionRCInterface.reserve(ResourceClient, Object) will throw a
35   * SecurityException if the application is not allowed to set-up a connection
36   * over the return channel at all (i.e., there is no valid target allowed).
37   */
