45  /** 
46   * <code>ServiceContextPermission</code> represents permission to
47   * control a <code>ServiceContext</code>.  A
48   * <code>ServiceContextPermission</code> contains a name (also
49   * referred to as a "target name") and an actions string.
50   *
51   * <p> The target name is the name of the service context permission
52   * (see the table below).  Each permission identifies a method.  A
53   * wildcard match is signified by an asterisk, i.e., "*".
54   *
55   * <p><a name="actions"></a> The actions string is either "own" or
56   * "*".
57   * A caller's "own" service contexts are those which it has created through
58   * {@link ServiceContextFactory#createServiceContext}.  In addition, an
59   * Xlet's "own" ServiceContext is the one in which it is currently running
60   * (see {@link ServiceContextFactory#getServiceContext(XletContext)}).
61   * The string "own" means
62   * the permission applies to your own service contexts; the string "*"
63   * implies permission to these, plus permission for service contexts
64   * obtained from all other sources.
65   *
66   * <p> The following table lists all the possible
67   * <code>ServiceContextPermission</code> target names, and describes
68   * what the permission allows for each.  <p>
69   *
70   * <table border=1 cellpadding=5>
71   * <tr>
72   * <th>Permission Target Name</th>
73   * <th>What the Permission Allows</th>
74   * </tr>
75   *
76   * <tr>
77   *    <td>access</td>
78   *    <td>Access to a <code>ServiceContext</code>, via <code>ServiceContextFactory.getServiceContexts()</code></td>
79   * </tr>
80   * 
81   * <tr>
82   *    <td>create</td>
83   *    <td>Creation of a <code>ServiceContext</code>.</td>
84   * </tr>
85   *
86   * <tr>
87   *    <td>destroy</td>
88   *    <td>Destruction of a <code>ServiceContext</code>.</td>
89   * </tr>
90   *
91   * <tr>
92   *    <td>getServiceContentHandlers</td>
93   *    <td>Obtaining the service content handlers from a <code>ServiceContext</code>.</td>
94   * </tr>
95   *
96   * <tr>
97   *    <td>stop</td>
98   *    <td>Stopping a <code>ServiceContext</code>.</td>
99   * </tr>
100  *
101  * </table>
102  *
103  * <p>
104  * The permission <code>ServiceContextPermission("access", "*")</code>

108  * {@link javax.microedition.xlet.XletContext} instance should only be
109  * accessible to another application if that other application has
110  * <code>ServiceContextPermission("access", "*")</code>.
111  * <p>
112  *
