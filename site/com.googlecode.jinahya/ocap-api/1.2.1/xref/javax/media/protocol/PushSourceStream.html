40  /** 
41   * Abstracts a read interface that pushes data.
42   *
43   *
44   * @see PushDataSource
45   * @version 1.7, 97/08/25.
46   */
47  public interface PushSourceStream extends SourceStream
48  {
49  
50      /** 
51       * Read from the stream without blocking.
52       * Returns -1 when the end of the media
53       * is reached.
54       *
55       * @param buffer The buffer to read bytes into.
56       * @param offset The offset into the buffer at which to begin writing data.
57       * @param length The number of bytes to read.
58       * @return The number of bytes read or -1
59       * when the end of stream is reached.
60       */
61      public int read(byte[] buffer, int offset, int length);
62  
63      /** 
64       * Determine the size of the buffer needed for the data transfer.
65       * This method is provided so that a transfer handler
66       * can determine how much data, at a minimum, will be
67       * available to transfer from the source.
68       * Overflow and data loss is likely to occur if this much
69       * data isn't read at transfer time.
70       *
71       * @return The size of the data transfer.
72       */
73      public int getMinimumTransferSize();
74  
75      /** 
76       * Register an object to service data transfers to this stream.
77       * <p>
78       * If a handler is already registered when
79       * <CODE>setTransferHandler</CODE> is called,
80       * the handler is replaced;
81       * there can only be one handler at a time.
82       * 
83       * @param transferHandler The handler to transfer data to.
84       */
85      public void setTransferHandler(SourceTransferHandler transferHandler);
86  }
