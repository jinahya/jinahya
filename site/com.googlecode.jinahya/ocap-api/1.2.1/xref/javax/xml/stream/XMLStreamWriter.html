28  /**
29   * The XMLStreamWriter interface specifies how to write XML.  The XMLStreamWriter  does
30   * not perform well-formedness checking on its input.  However
31   * the writeCharacters method is required to escape &amp; , &lt; and &gt;
32   * For attribute values the writeAttribute method will escape the 
33   * above characters plus &quot; to ensure that all character content
34   * and attribute values are well formed. 
35   *
36   * Each NAMESPACE 
37   * and ATTRIBUTE must be individually written.
38   *
39   * If javax.xml.stream.isRepairingNamespaces is set to <code>false</code> it is a fatal error if an element
40   * is written with namespace URI that has not been bound to a prefix.
41   *
42   * If javax.xml.stream.isRepairingNamespaces is set to <code>true</code> the XMLStreamWriter implementation
43   * must write a prefix for each unbound URI that it encounters in the
44   * current scope.  
45   *
46   * @version 1.0
47   * @author Copyright (c) 2003 by BEA Systems. All Rights Reserved.
48   * @see XMLOutputFactory
49   * @see XMLStreamReader
50   */

58     * @throws XMLStreamException
