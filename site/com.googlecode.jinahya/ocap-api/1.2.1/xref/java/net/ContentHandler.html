41  /** 
42   * The abstract class <code>ContentHandler</code> is the superclass 
43   * of all classes that read an <code>Object</code> from a 
44   * <code>URLConnection</code>. 
45   * <p>
46   * An application does not generally call the 
47   * <code>getContent</code> method in this class directly. Instead, an 
48   * application calls the <code>getContent</code> method in class 
49   * <code>URL</code> or in <code>URLConnection</code>.
50   * The application's content handler factory (an instance of a class that 
51   * implements the interface <code>ContentHandlerFactory</code> set 
52   * up by a call to <code>setContentHandler</code>) is 
53   * called with a <code>String</code> giving the MIME type of the 
54   * object being received on the socket. The factory returns an 
55   * instance of a subclass of <code>ContentHandler</code>, and its 
56   * <code>getContent</code> method is called to create the object. 
57   * <p>
58   * If no content handler could be found, URLConnection will 
59   * look for a content handler in a user-defineable set of places.
60   * By default it looks in sun.net.www.content, but users can define a 
61   * vertical-bar delimited set of class prefixes to search through in 
62   * addition by defining the java.content.handler.pkgs property.
63   * The class name must be of the form:
64   * <pre>
65   *     {package-prefix}.{major}.{minor}
66   * e.g.
67   *     YoyoDyne.experimental.text.plain
68   * </pre>
69   * If the loading of the content handler class would be performed by 
70   * a classloader that is outside of the the delegation chain of the caller,
71   * the JVM will need the RuntimePermission "getClassLoader".
72   * 
73   * @author  James Gosling
74   * @version 1.14, 02/02/00
75   * @see     java.net.ContentHandler#getContent(java.net.URLConnection)
76   * @see     java.net.ContentHandlerFactory
77   * @see     java.net.URL#getContent()
78   * @see     java.net.URLConnection
79   * @see     java.net.URLConnection#getContent()
80   * @see     java.net.URLConnection#setContentHandlerFactory(java.net.ContentHandlerFactory)
81   * @since   JDK1.0
82   */
83  public abstract class ContentHandler
84  {
85  
86      public ContentHandler() { }

88      /** 
89       * Given a URL connect stream positioned at the beginning of the 
90       * representation of an object, this method reads that stream and 
91       * creates an object from it. 
92       *
93       * @param      urlc   a URL connection.
94       * @return     the object read by the <code>ContentHandler</code>.
95       * @exception  IOException  if an I/O error occurs while reading the object.
96       */
97      public abstract Object getContent(URLConnection urlc) throws IOException;
