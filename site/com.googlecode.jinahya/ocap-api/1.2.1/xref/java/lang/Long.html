261      * Parses the string argument as a signed <code>long</code> in the
262      * radix specified by the second argument. The characters in the
263      * string must all be digits of the specified radix (as determined
264      * by whether {@link java.lang.Character#digit(char, int)} returns
265      * a nonnegative value), except that the first character may be an
266      * ASCII minus sign <code>'-'</code> (<code>'&#92;u002D'</code>) to
267      * indicate a negative value. The resulting <code>long</code>
268      * value is returned.
269      * <p>
270      * Note that neither the character <code>L</code>
271      * (<code>'&#92;u004C'</code>) nor <code>l</code>
272      * (<code>'&#92;u006C'</code>) is permitted to appear at the end
273      * of the string as a type indicator, as would be permitted in
274      * Java programming language source code - except that either
275      * <code>L</code> or <code>l</code> may appear as a digit for a
276      * radix greater than 22.
277      * <p>
278      * An exception of type <code>NumberFormatException</code> is
279      * thrown if any of the following situations occurs:
280      * <ul>
281      * <li>The first argument is <code>null</code> or is a string of
282      * length zero.
283      * <li>The <code>radix</code> is either smaller than {@link
284      * java.lang.Character#MIN_RADIX} or larger than {@link
285      * java.lang.Character#MAX_RADIX}.
286      * <li>Any character of the string is not a digit of the specified
287      * radix, except that the first character may be a minus sign
288      * <code>'-'</code> (<code>'&#92;u002d'</code>) provided that the
289      * string is longer than length 1.
290      * <li>The value represented by the string is not a value of type
291      *      <code>long</code>. 
292      * </ul><p>
293      * Examples:
294      * <blockquote><pre>
295      * parseLong("0", 10) returns 0L
296      * parseLong("473", 10) returns 473L
297      * parseLong("-0", 10) returns 0L
298      * parseLong("-FF", 16) returns -255L
299      * parseLong("1100110", 2) returns 102L
300      * parseLong("99", 8) throws a NumberFormatException
301      * parseLong("Hazelnut", 10) throws a NumberFormatException
302      * parseLong("Hazelnut", 36) returns 1356099454469L
303      * </pre></blockquote>
304      * 
305      * @param      s       the <code>String</code> containing the
306      *                     <code>long</code> representation to be parsed.
307      * @param      radix   the radix to be used while parsing <code>s</code>.
308      * @return     the <code>long</code> represented by the string argument in
309      *             the specified radix.
310      * @exception  NumberFormatException  if the string does not contain a
311      *               parsable <code>long</code>.
