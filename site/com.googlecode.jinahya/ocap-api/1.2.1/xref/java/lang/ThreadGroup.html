41  /** 
42   * A thread group represents a set of threads. In addition, a thread 
43   * group can also include other thread groups. The thread groups form 
44   * a tree in which every thread group except the initial thread group 
45   * has a parent. 
46   * <p>
47   * A thread is allowed to access information about its own thread 
48   * group, but not to access information about its thread group's 
49   * parent thread group or any other thread groups. 
50   *
51   * @author  unascribed
52   * @version 1.55 07/27/01
53   * @since   JDK1.0
54   */
