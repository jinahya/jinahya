118     /** 
119      * Returns true if this CodeSource object "implies" the specified CodeSource.
120      * <P>
121      * More specifically, this method makes the following checks, in order. 
122      * If any fail, it returns false. If they all succeed, it returns true.<p>
123      * <ol>
124      * <li> <i>codesource</i> must not be null.
125      * <li> If this object's certificates are not null, then all
126      * of this object's certificates must be present in <i>codesource</i>'s 
127      * certificates.
128      * <li> If this object's location (getLocation()) is not null, then the 
129      * following checks are made against this object's location and 
130      * <i>codesource</i>'s:<p>
131      *   <ol>
132      *     <li>  <i>codesource</i>'s location must not be null.
133      *
134      *     <li>  If this object's location 
135      *           equals <i>codesource</i>'s location, then return true.
136      *
137      *     <li>  This object's protocol (getLocation().getProtocol()) must be
138      *           equal to <i>codesource</i>'s protocol.
139      *
140      *     <li>  If this object's host (getLocation().getHost()) is not null,  
141      *           then the SocketPermission
142      *           constructed with this object's host must imply the
143      *           SocketPermission constructed with <i>codesource</i>'s host.
144      *
145      *     <li>  If this object's port (getLocation().getPort()) is not 
146      *           equal to -1 (that is, if a port is specified), it must equal 
147      *           <i>codesource</i>'s port.
148      *
149      *     <li>  If this object's file (getLocation().getFile()) doesn't equal
150      *           <i>codesource</i>'s file, then the following checks are made:
151      *           If this object's file ends with "/-",
152      *           then <i>codesource</i>'s file must start with this object's
153      *           file (exclusive the trailing "-").
154      *           If this object's file ends with a "/*",
155      *           then <i>codesource</i>'s file must start with this object's
156      *           file and must not have any further "/" separators.
157      *           If this object's file doesn't end with a "/", 
158      *           then <i>codesource</i>'s file must match this object's 
159      *           file with a '/' appended.
160      *
161      *     <li>  If this object's reference (getLocation().getRef()) is 
162      *           not null, it must equal <i>codesource</i>'s reference.
163      *
164      *   </ol>
165      * </ol>
166      * <p>
167      * For example, the codesource objects with the following locations
168      * and null certificates all imply
169      * the codesource with the location "http://java.sun.com/classes/foo.jar"
170      * and null certificates:
171      * <pre>
172      *     http:
173      *     http://*.sun.com/classes/*
174      *     http://java.sun.com/classes/-
175      *     http://java.sun.com/classes/foo.jar
176      * </pre>
177      * 
178      * Note that if this CodeSource has a null location and a null
179      * certificate chain, then it implies every other CodeSource.
180      *
181      * @param codesource CodeSource to compare against.
182      *
183      * @return true if the specified codesource is implied by this codesource,
184      * false if not.  
185      */
