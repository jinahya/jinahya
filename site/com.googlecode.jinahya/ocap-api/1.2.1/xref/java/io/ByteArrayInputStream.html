39  /** 
40   * A <code>ByteArrayInputStream</code> contains
41   * an internal buffer that contains bytes that
42   * may be read from the stream. An internal
43   * counter keeps track of the next byte to
44   * be supplied by the <code>read</code> method.
45   * <p>
46   * Closing a <tt>ByteArrayInputStream</tt> has no effect. The methods in
47   * this class can be called after the stream has been closed without
48   * generating an <tt>IOException</tt>.
49   *
50   * @author  Arthur van Hoff

56      /** 
57       * An array of bytes that was provided

59       * through <code>buf[count-1]</code> are the
60       * only bytes that can ever be read from the
61       * stream;  element <code>buf[pos]</code> is
62       * the next byte to be read.
63       */
