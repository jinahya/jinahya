40   * A <code>PrintStream</code> adds functionality to another output stream,
41   * namely the ability to print representations of various data values
42   * conveniently.  Two other features are provided as well.  Unlike other output
43   * streams, a <code>PrintStream</code> never throws an
44   * <code>IOException</code>; instead, exceptional situations merely set an
45   * internal flag that can be tested via the <code>checkError</code> method.
46   * Optionally, a <code>PrintStream</code> can be created so as to flush
47   * automatically; this means that the <code>flush</code> method is
48   * automatically invoked after a byte array is written, one of the
49   * <code>println</code> methods is invoked, or a newline character or byte
50   * (<code>'\n'</code>) is written.
51   *
52   * <p> All characters printed by a <code>PrintStream</code> are converted into
53   * bytes using the platform's default character encoding.  The <code>{@link
54   * PrintWriter}</code> class should be used in situations that require writing
55   * characters rather than bytes.
56   *
57   * @version    1.21, 00/02/02
