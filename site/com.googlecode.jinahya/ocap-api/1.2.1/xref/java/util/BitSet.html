41  /** 
42   * This class implements a vector of bits that grows as needed. Each 
43   * component of the bit set has a <code>boolean</code> value. The 
44   * bits of a <code>BitSet</code> are indexed by nonnegative integers. 
45   * Individual indexed bits can be examined, set, or cleared. One 
46   * <code>BitSet</code> may be used to modify the contents of another 
47   * <code>BitSet</code> through logical AND, logical inclusive OR, and 
48   * logical exclusive OR operations.
49   * <p>
50   * By default, all bits in the set initially have the value 
51   * <code>false</code>. 
52   * <p>
53   * Every bit set has a current size, which is the number of bits 
54   * of space currently in use by the bit set. Note that the size is
55   * related to the implementation of a bit set, so it may change with
56   * implementation. The length of a bit set relates to logical length
57   * of a bit set and is defined independently of implementation.
58   * <p>
59   * Unless otherwise noted, passing a null parameter to any of the
60   * methods in a <code>BitSet</code> will result in a
61   * <code>NullPointerException</code>.
62   *
63   * A <code>BitSet</code> is not safe for multithreaded use without
64   * external synchronization.
65   *
