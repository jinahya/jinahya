42  /** 
43   * The <code>IndexColorModel</code> class is a <code>ColorModel</code>
44   * class that works with pixel values consisting of a
45   * single sample that is an index into a fixed colormap in the default
46   * sRGB color space.  The colormap specifies red, green, blue, and
47   * optional alpha components corresponding to each index.  All components
48   * are represented in the colormap as 8-bit unsigned integral values.
49   * Some constructors allow the caller to specify "holes" in the colormap
50   * by indicating which colormap entries are valid and which represent
51   * unusable colors via the bits set in a <code>BigInteger</code> object.
52   * This color model is similar to an X11 PseudoColor visual.
53   * <p>
54   * Some constructors provide a means to specify an alpha component
55   * for each pixel in the colormap, while others either provide no
56   * such means or, in some cases, a flag to indicate whether the
57   * colormap data contains alpha values.  If no alpha is supplied to
58   * the constructor, an opaque alpha component (alpha = 1.0) is
59   * assumed for each entry.
60   * An optional transparent pixel value can be supplied that indicates a
61   * completely transparent pixel, regardless of any alpha component
62   * supplied or assumed for that pixel value.
63   * Note that the color components in the colormap of an
64   * <code>IndexColorModel</code> objects are never pre-multiplied with
65   * the alpha components.
66   * <p>
67   * <a name="transparency">
68   * The transparency of an <code>IndexColorModel</code> object is
69   * determined by examining the alpha components of the colors in the
70   * colormap and choosing the most specific value after considering
71   * the optional alpha values and any transparent index specified.
72   * The transparency value is <code>Transparency.OPAQUE</code>
73   * only if all valid colors in
74   * the colormap are opaque and there is no valid transparent pixel.
75   * If all valid colors
76   * in the colormap are either completely opaque (alpha = 1.0) or
77   * completely transparent (alpha = 0.0), which typically occurs when
78   * a valid transparent pixel is specified,
79   * the value is <code>Transparency.BITMASK</code>.
80   * Otherwise, the value is <code>Transparency.TRANSLUCENT</code>, indicating
81   * that some valid color has an alpha component that is
82   * neither completely transparent nor completely opaque (0.0 < alpha < 1.0).
83   * </a>
84   * <p>
85   * The index represented by a pixel value is stored in the least
86   * significant <em>n</em> bits of the pixel representations passed to the
87   * methods of this class, where <em>n</em> is the pixel size specified to the
88   * constructor for a particular <code>IndexColorModel</code> object;
89   * <em>n</em> must be between 1 and 16, inclusive.  
90   * Higher order bits in pixel representations are assumed to be zero.
91   * For those methods that use a primitive array pixel representation of
92   * type <code>transferType</code>, the array length is always one.  
93   * The transfer types supported are <code>DataBuffer.TYPE_BYTE</code> and 
94   * <code>DataBuffer.TYPE_USHORT</code>.  A single int pixel 
95   * representation is valid for all objects of this class, since it is 
96   * always possible to represent pixel values used with this class in a 
97   * single int.  Therefore, methods that use this representation do 
98   * not throw an <code>IllegalArgumentException</code> due to an invalid 
99   * pixel value.
100  * <p>
101  * Many of the methods in this class are final.  The reason for
102  * this is that the underlying native graphics code makes assumptions
103  * about the layout and operation of this class and those assumptions
104  * are reflected in the implementations of the methods here that are
105  * marked final.  You can subclass this class for other reaons, but
106  * you cannot override or modify the behaviour of those methods.
107  *
108  * @see ColorModel
109  * @see ColorSpace
110  * @see DataBuffer
111  *
112  * @version 10 Feb 1997
113  */
