43  /** 
44   * An ImageFilter class for scaling images using a simple area averaging
45   * algorithm that produces smoother results than the nearest neighbor
46   * algorithm.
47   * <p>This class extends the basic ImageFilter Class to scale an existing
48   * image and provide a source for a new image containing the resampled
49   * image.  The pixels in the source image are blended to produce pixels
50   * for an image of the specified size.  The blending process is analogous
51   * to scaling up the source image to a multiple of the destination size
52   * using pixel replication and then scaling it back down to the destination
53   * size by simply averaging all the pixels in the supersized image that
54   * fall within a given pixel of the destination image.  If the data from
55   * the source is not delivered in TopDownLeftRight order then the filter
56   * will back off to a simple pixel replication behavior and utilize the
57   * requestTopDownLeftRightResend() method to refilter the pixels in a
58   * better way at the end.
59   * <p>It is meant to be used in conjunction with a FilteredImageSource
60   * object to produce scaled versions of existing images.  Due to
61   * implementation dependencies, there may be differences in pixel values 
62   * of an image filtered on different platforms.
63   *
64   * @see FilteredImageSource
65   * @see ReplicateScaleFilter
66   * @see ImageFilter

70   */
