39   * The <code>GridLayout</code> class is a layout manager that 
40   * lays out a container's components in a rectangular grid. 
41   * The container is divided into equal-sized rectangles, 
42   * and one component is placed in each rectangle. 
43   * For example, the following is an applet that lays out six buttons 
44   * into three rows and two columns: 
45   *
46   * <em><p>Note: The following code example includes classes that do
47   * not appear in this specification. Their inclusion is purely to
48   * serve as a demonstration.</em>
49   *
50   * <p>
51   * <hr><blockquote>
52   * <pre>
53   * import java.awt.*;
54   * import java.applet.Applet;
55   * public class ButtonGrid extends Applet {
56   *     public void init() {
57   *         setLayout(new GridLayout(3,2));
58   *         add(new Button("1"));
59   *         add(new Button("2"));
60   *         add(new Button("3"));
61   *         add(new Button("4"));
62   *         add(new Button("5"));
63   *         add(new Button("6"));
64   *     }
65   * }
66   * </pre></blockquote><hr>     
67   * <p>
68   * It produces the following output:
69   * <p>
70   * <center><table COLS=2 WIDTH=600 summary="layout">
71   * <tr ALIGN=CENTER>
72   * <td><img SRC="doc-files/GridLayout-1.gif"
73   *      alt="Shows 6 buttons in rows of 2. Row 1 shows buttons 1 then 2.
74   * Row 2 shows buttons 3 then 4. Row 3 shows buttons 5 then 6.">
75   * </td>
76   * 
77   * </tr>
78   * <!-- PBP/PP 6187225 --!>
79   * 
80   * </table></center>
81   * <p>
82   * When both the number of rows and the number of columns have 
83   * been set to non-zero values, either by a constructor or 
84   * by the <tt>setRows</tt> and <tt>setColumns</tt> methods, the number of 
85   * columns specified is ignored.  Instead, the number of 
86   * columns is determined from the specified number or rows 
87   * and the total number of components in the layout. So, for 
88   * example, if three rows and two columns have been specified 
89   * and nine components are added to the layout, they will 
90   * be displayed as three rows of three columns.  Specifying 
91   * the number of columns affects the layout only when the 
92   * number of rows is set to zero.
93   *
