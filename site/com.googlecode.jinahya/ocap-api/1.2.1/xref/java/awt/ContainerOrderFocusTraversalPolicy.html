39   * A FocusTraversalPolicy that determines traversal order based on the order
40   * of child Components in a Container. From a particular focus cycle root, the
41   * policy makes a pre-order traversal of the Component hierarchy, and traverses
42   * a Container's children according to the ordering of the array returned by
