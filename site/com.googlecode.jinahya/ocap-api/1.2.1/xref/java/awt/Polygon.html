47  /** 
48   * The <code>Polygon</code> class encapsulates a description of a 
49   * closed, two-dimensional region within a coordinate space. This 
50   * region is bounded by an arbitrary number of line segments, each of 
51   * which is one side of the polygon. Internally, a polygon
52   * comprises of a list of (<i>x</i>,&nbsp;<i>y</i>) 
53   * coordinate pairs, where each pair defines a <i>vertex</i> of the 
54   * polygon, and two successive pairs are the endpoints of a 
55   * line that is a side of the polygon. The first and final
56   * pairs of (<i>x</i>,&nbsp;<i>y</i>) points are joined by a line segment 
57   * that closes the polygon.  This <code>Polygon</code> is defined with
58   * an even-odd winding rule.
59  
60  <em>(The even-odd rule specifies that a point lies inside the  path if a ray drawn in any direction from that point to  infinity is crossed by path segments an odd number of times.)
61  </em>
62   * This class's hit-testing methods, which include the 
63   * <code>contains</code>, <code>intersects</code> and <code>inside</code>
64   * methods, use the <i>insideness</i> definition described in the 
65   * {@link Shape} class comments.
66   *
67   * @version     1.26, 07/24/98
68   * @author 	Sami Shaio
