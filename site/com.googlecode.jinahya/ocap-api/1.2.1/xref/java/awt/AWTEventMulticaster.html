47  /** 
48   * A class which implements efficient and thread-safe multi-cast event 
49   * dispatching for the AWT events defined in the java.awt.event package.  
50   * This class will manage an immutable structure consisting of a chain of 
51   * event listeners and will dispatch events to those listeners.  Because
52   * the structure is immutable, it is safe to use this API to add/remove
53   * listeners during the process of an event dispatch operation.
54   * However, event listeners added during the process of an event dispatch 
55   * operation will not be notified of the event currently being dispatched.
56   *
57   * An example of how this class could be used to implement a new
58   * component which fires "action" events:
59   *
60   * <pre><code>
61   * public myComponent extends Component {
62   *     ActionListener actionListener = null;
63   *
64   *     public synchronized void addActionListener(ActionListener l) {
65   *	   actionListener = AWTEventMulticaster.add(actionListener, l);
66   *     }
67   *     public synchronized void removeActionListener(ActionListener l) {
68   *  	   actionListener = AWTEventMulticaster.remove(actionListener, l);
69   *     }
70   *     public void processEvent(AWTEvent e) {
71   *         // when event occurs which causes "action" semantic
72   *         ActionListener listener = actionListener;
73   *         if (listener != null) {
74   *             listener.actionPerformed(new ActionEvent());
75   *         }
76   *     }
77   * }
78   * </code></pre>
79   *
80   * @author      John Rose
81   * @author 	Amy Fowler
82   * @version 	1.35, 01/23/03
83   * @since 	1.1
