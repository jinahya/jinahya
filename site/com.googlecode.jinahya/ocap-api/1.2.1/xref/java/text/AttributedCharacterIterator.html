45  /** 
46   * An AttributedCharacterIterator allows iteration through both text and
47   * related attribute information.
48   *
49   * <p>
50   * An attribute is a key/value pair, identified by the key.  No two
51   * attributes on a given character can have the same key.
52   *
53   * <p>The values for an attribute are immutable, or must not be mutated
54   * by clients or storage.  They are always passed by reference, and not
55   * cloned.
56   *
57   * <p>A <em>run with respect to an attribute</em> is a maximum text range for
58   * which:
59   * <ul>
60   * <li>the attribute is undefined or null for the entire range, or
61   * <li>the attribute value is defined and has the same non-null value for the
62   *     entire range.
63   * </ul>
64   *
65   * <p>A <em>run with respect to a set of attributes</em> is a maximum text range for
66   * which this condition is met for each member attribute.
67   *
68   * <p>The returned indexes are limited to the range of the iterator.
69   *
70   * <p>The returned attribute information is limited to runs that contain
71   * the current character.
